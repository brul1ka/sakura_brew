# SakuraBrew

Добро пожаловать в **SakuraBrew** - текстовую симуляцию управления японской кофейней! 
В этой игре вам предстоит вести свой бизнес, управлять запасами кофе, устанавливать цены и справляться с неожиданными событиями, чтобы не разориться за 10 игровых дней.
Был задуман как практика в начинаниях познания кодинга.

---

## Как играть
1. Качаем зависимости:
```bash
pip install -r requirements.txt
```
2. Запускаем игру:  
```bash
python SakuraBrew.py
```
**или**
```bash
python SakuraBrew.py
```
